# Hello World.
How to print "Hello World" in multiple programming languages.
```C
// Code
#include <stdio.h>
int main() {
   printf("Hello, World!");
   return 0;
}
```
```C++
// CODE
#include <iostream>

int main() {
    std::cout << "Hello World!";
    return 0;
}
```
```python
# CODE
print("Hello World")
```
```java
// CODE
public static void main(String[] args){
    System.out.println("Hello World");
}
```
